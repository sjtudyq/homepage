Research Experience

Exploring Non-iid Scenario in Federated Learning Jun 2020 - Present
Research Assistant, supervised by Professor Bingsheng He, NUS

    We summarized the non-iid data partition in previous works, and added more data partition methods based on project FedMA. We are currently running a benchmark for different non-iid scenario using FedAvg algorithm. The algorithm fails in extreme non-iid case (e.g. each party only contains one category of data). Still on progress to solve this problem.

Learning to Classify Carotid Artery Severity Sep 2019 - Jan 2020
Independent Research, supervised by Professor Chandrajit Bajaj, UT Austin

    This study uses various methods to classify the severity of carotid artery. We revised the Augmented Neural Ode Net for this medical task and slightly improved the accuracy to ~80%. With some domain knowledge, we know the data contains much noise and probably fail to yield accuracy above 90% (which could be put in practical use then). Details can be found in this abstract. How much test accuracy can we achieve on a data set? Is our model complex enough? These are all open questions.

Learning to Classify 3D Mesh Jul 2019 - Jun 2020
Research Assistant, supervised by Professor Cewu Lu, SJTU

    This study aims to use new structures to extract 3D mesh information. Meshes are stored in graph structure. It is still an open question how to effectively extract features in graph. We published a paper to recover 3D hand mesh from RGB image, as an application for robotics.
